My Journey in IT: From Basics to Building Real-World Applications

Russel Cabigquez

Jan 27, 2025

Hi, I'm an IT student at PLV, currently in my second year, and my journey in technology has been an exciting one so far. I started with little to no programming knowledge, but over time, I’ve explored and learned various technologies that are shaping my skills and future career.

Discovering the World of Programming

My programming journey began with the basics—learning HTML, CSS, and Javascript. These languages form the foundation of web development, and understanding them was crucial for me. At first, it felt overwhelming, especially trying to grasp how everything connects to create functional web pages. With practice and dedication, I managed to build small projects, such as basic websites and interactive components.

After taking a break from web development, I noticed that some of the concepts felt unfamiliar again. To address this, I started reviewing the fundamentals, focusing on key aspects like responsive design, DOM manipulation, and integrating CSS for user-friendly designs. This review not only refreshed my memory but also gave me more confidence to dive deeper into more complex topics.

Diving Into Full-Stack Development

As I progressed, I expanded my knowledge by learning the MERN stack (MongoDB, Express, React, Node.js). Each technology brought something unique to the table. MongoDB introduced me to databases and how to store and retrieve data efficiently. Express.js taught me the importance of creating server-side logic and handling API requests, which is an essential part of any modern web application.

React, on the other hand, completely transformed how I viewed front-end development. Building reusable components and working with state management opened new doors for creating dynamic user interfaces. With Node.js, I understood how JavaScript could be extended to work seamlessly on the server side, making it possible to write full-stack applications using a single programming language.

Currently, I'm focusing on learning authentication techniques in Express.js to create secure and user-friendly applications. Security is a vital part of any software, and I want to ensure that the applications I build are both robust and reliable.

Building a Real-World Project

One of the most significant milestones in my journey has been creating an Airport Management System, a desktop application developed using C# and WPF. This project was born out of my interest in combining user-friendly interfaces with efficient backend logic.

Designing the application taught me how to structure complex software solutions. Using WPF and XAML for the user interface, I learned how to position elements, handle events, and manage resources such as images and fonts. A notable challenge I faced was dealing with PNG images that had transparency issues, where the transparent backgrounds appeared filled. Solving this required exploring image formats and ensuring compatibility with WPF's rendering engine.

I also experimented with global object instances to improve the app’s efficiency. This approach allowed me to reuse objects across different modules without the overhead of creating new instances, ensuring better performance and memory usage.

Another exciting aspect was designing a responsive UI. For example, I worked on ensuring that the border below the dashboard expanded correctly when resizing the window. These small adjustments taught me how important it is to pay attention to the details of user interaction.

My Goals Moving Forward

Looking ahead, I’m excited to continue learning and building on the foundation I’ve established. My goals include:

  • Mastering authentication techniques in Express.js and exploring advanced backend development concepts.
  • Strengthening my front-end development skills by diving deeper into React and using tools like Tailwind CSS to create visually appealing and responsive designs.
  • Exploring new technologies, frameworks, and tools that can enhance my problem-solving skills and efficiency.

This journey has been one of growth and discovery, and I’m eager to see how I can continue to evolve as a developer. Every project I take on teaches me something new, and I’m excited for the challenges and opportunities that lie ahead!
